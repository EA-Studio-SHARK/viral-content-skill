# viral-content

中文社媒爆款内容方法论。让 AI 懂平台算法、钩子、痛点开头、互动结尾。

## Use When

- 需要写小红书笔记、抖音脚本、视频号脚本或公众号文章。
- 普通 AI 输出太平、太像广告、太像作文。
- 需要把一个主题改写成更有点击率、完播率、收藏率、评论率的内容。
- 需要 Before / After 对比、标题优化、封面钩子、话题标签或互动结尾。
