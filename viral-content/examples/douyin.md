# 抖音 Before / After：理财入门

## Before：AI 普通输出

```text
标题：理财入门知识

大家好，今天给大家介绍一些理财入门知识。理财首先要学会记账，其次要控制消费，然后可以了解基金、股票等投资方式。理财需要长期坚持，大家要根据自己的风险承受能力选择适合自己的产品。
```

问题：

- 开头没有黄金3秒，用户会直接划走。
- 没有画面和口播节奏。
- 内容太像课程摘要，没有冲突和行动感。
- 结尾没有关注理由。

## After：使用 viral-content skill

```text
标题：你不是赚得少，是钱漏掉了

镜头1｜画面：手机账单快速闪过，外卖、奶茶、打车金额放大
口播：你以为自己存不下钱，是因为工资太低？
字幕重点：不是工资低，是钱在偷偷漏

镜头2｜画面：博主拿出一张纸，写下“收入 - 固定支出 - 乱花钱”
口播：新手理财第一步，不是买基金，也不是研究股票，而是先抓住这3个数字。
字幕重点：新手别先碰投资

镜头3｜画面：屏幕出现三栏表格
口播：每个月发工资后，先把房租、通勤、吃饭这些固定支出扣掉，再看你真正能支配多少钱。
字幕重点：先算可支配收入

镜头4｜画面：奶茶、会员、冲动购物被红圈标出
口播：然后找出最容易失控的3个消费项。很多人的第一笔存款，就是从少点外卖、少买闲置开始的。
字幕重点：先堵漏洞，再谈增值

镜头5｜画面：存钱进度条从0到1000元
口播：你先别急着赚收益，先让账户里稳定多出1000块，这才是理财入门。
字幕重点：先存下1000

镜头6｜画面：博主看向镜头
口播：下一条我讲，月薪5000的人怎么分配工资。关注我，别等钱花完了才想起来学理财。
字幕重点：关注看工资分配法
```

风险提示：

```text
内容为个人理财习惯分享，不构成投资建议。任何投资都有风险，先学习，再决策。
```
