# viral-content-skill
> Make your AI write Chinese social content that can actually go viral.

![License](https://img.shields.io/badge/license-MIT-blue.svg)

`viral-content-skill` is an AI Agent Skill for Claude Code, Cursor, Codex, and similar agents. It helps creators generate Chinese social content for Xiaohongshu, Douyin, WeChat Channels, and WeChat Official Accounts using platform-aware hooks, pain-point openings, algorithm-friendly structure, and interaction-driven endings.

## Install

```bash
npx skills add EA-Studio-SHARK/viral-content-skill
```

## Platforms

Xiaohongshu / Douyin / WeChat Channels / WeChat Official Accounts.

## Rule Preview

- Every title needs a hook: curiosity, benefit, contrast, or identity.
- The first 3 seconds or first sentence must hit a pain point or open a loop.
- Write with direct "you" language so the reader feels personally called out.
- Use concrete scenes, numbers, and actions instead of abstract claims.
- Always end with an interaction prompt: comment, save, share, or follow.
